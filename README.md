# python-helt-fra-bunden

Dette pythonkursus henvender sig til elever i folkeskolen, der gerne vil i gang med at lære at kode python. 

Kurset benytter sig af Google Colab, hvilket betyder at der ikke skal installeres noget på elevernes computere, og man kan arbejde direkte i en browser.

Kurset er delt ind i blokke, som skal løses i rækkefølge. 
Klik på linket under hvert element, for at åbne modulet i browseren.
