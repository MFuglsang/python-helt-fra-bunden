## 💻 Arbejd Lokalt på Din Computer

Du kan vælge at arbejde på din egen computer i stedet for Google Colab. Her er vejledningen:

### 🎯 Mulighed 1: Google Colab (Nemmest - Ingen Installation)

**Fordele:**
- ✅ Intet at installere
- ✅ Virker på enhver computer
- ✅ Dine filer gemmes automatisk på Google Drive
- ✅ Perfekt til at starte

**Hvordan:**
1. Klik på linket til det modul du vil arbejde med (ovenfor)
2. En Colab-notesbog åbner i din browser
3. Klik på celle og tryk `Shift+Enter` for at køre kode
4. Alt gemmes automatisk!

---

### 💾 Mulighed 2: Lokal Installation (Mere Kontrol)

Hvis du vil arbejde direkte på din computer, er her vejledningen:

#### **Trin 1: Installer Git**

Git bruges til at hente hele projektet fra GitHub.

**Windows/Mac/Linux:**
- Gå til https://git-scm.com/download
- Download og installer
- Åbn Terminal (Mac/Linux) eller Command Prompt (Windows) og skriv:
  ```bash
  git --version
  ```
  Hvis det virker, har du Git installeret ✅

#### **Trin 2: Klone Projektet**

1. Åbn Terminal/Command Prompt
2. Naviger til mappen hvor du vil have projektet:
   ```bash
   cd Desktop
   ```
   (eller hvor du ønsker det)

3. Hent hele projektet:
   ```bash
   git clone https://github.com/MFuglsang/python-helt-fra-bunden.git
   ```

4. Gå ind i mappen:
   ```bash
   cd python-helt-fra-bunden
   ```

#### **Trin 3: Installer Python og Jupyter**

**Option A: Med Anaconda (Nemmest)**

1. Download Anaconda fra https://www.anaconda.com/download
2. Installer det
3. Åbn Anaconda Navigator
4. Søg efter "Jupyter Notebook" og klik "Launch"
5. Jupyter åbner i din browser
6. Naviger til `notebooks/` mappen
7. Klik på en .ipynb fil for at åbne den

**Option B: Med pip (Hvis du allerede har Python)**

1. Åbn Terminal/Command Prompt
2. Installer Jupyter:
   ```bash
   pip install jupyter
   ```

3. Start Jupyter:
   ```bash
   jupyter notebook
   ```

4. Jupyter åbner i din browser
5. Naviger til `notebooks/` mappen
6. Klik på en .ipynb fil

#### **Trin 4: Arbejd med Notebooken**

1. En celle med kode ser sådan ud:
   ```python
   print('Hej verden')
   ```

2. For at køre cellen:
   - Klik på cellen
   - Tryk `Shift+Enter`
   - Resultatet vises under cellen

3. Edit cellen, køre den igen, osv.

#### **Trin 5: Gem Dit Arbejde**

Jupyter gemmer automatisk dine ændringer når du arbejder. Men for at være sikker:

1. Tryk `Ctrl+S` (Windows) eller `Cmd+S` (Mac)
2. Eller menuen: File → Save and Checkpoint

**Vigtigt:** Dine ændringer gemmes kun på DIN computer!

---

### 🎨 Mulighed 3: VS Code (For dem der elsker Editors)

**Spørgsmål: Hvor længe tager kurset?**
