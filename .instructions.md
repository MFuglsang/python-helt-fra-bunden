# Python helt fra bunden - VS Code Instructions

Du arbejder på **Python helt fra bunden**, et kodningskursus til danske folkeskoleelever.

## Hurtigt Overblik

- **Målgruppe**: 7.-9. klasse elever uden programmerings-erfaring
- **Format**: Jupyter Notebooks via Google Colab
- **Sprog**: Dansk
- **Struktur**: 3+ moduler (Intro → Variable → Betingelser → ...)

## Når du udvikler nyt materiale

### Vigtigste Regler

1. **"Ville en 12-årig uden kodeerfaringer forstå det?"**
   - Hvis JA → det er klart nok
   - Hvis NEJ → simplificer

2. **Tone**: Venlig, opmuntrende, "vi" ikke "I"
   - Start med "Hej, fremtidige utviklere!"
   - Validér at fejl er OK

3. **Eksempler før øvelser**
   - Eleverne ser først kode der virker
   - Så løser de selv

4. **Korte kodeceller**
   - Max 20-30 linjer per celle
   - En celle = en ide

### Notebook-struktur for nyt modul

```
notebooks/
NN_emnenavn.ipynb

Indhold:
1. Markdown: Introduktion (hvad + hvorfor)
2. Code: Eksempel 1 (super simpelt)
3. Markdown: Teori for koncept 1
4. Code: Eksempel 2 (udviklelse)
5. Code: Øvelse 1 ("Øvelse: ...")
... [gentag for flere koncepter]
```

### Naming Convention

- Filnavn: `NN_emnenavn.ipynb` (f.eks. `04_løkker.ipynb`)
- Sproget: **DANSK** i filnavn
- Nummer: For sortering og rækkefølge

### Før du udgiver

- [ ] Kør alle codeceller - de skal virke
- [ ] Test links til Google Colab virker
- [ ] Læs for stavefejl og klarhed
- [ ] Tilføj til README.md med githubtocolab link
- [ ] Lav løsningsnotebook eller opdater lærervejledning.md
- [ ] Spørg dig selv: "Kan eleverne gøre det på 2 minutter?"

## Fælles Mønstre du skal følge

### ✅ Godt: Kode med Vejledning
```python
# Eksempel: Gemme et navn i en variabel
navn = 'Maria'
print(navn)

# Øvelse: Prøv med dit navn
```

### ✅ Godt: Markdown med Historie
```markdown
## Hvad er en løkke?

Forestil dig at du skal tegne 100 stjerner...
En løkke er som at sige: "Gentag dette 100 gange!"
```

### ❌ Dårligt: For Teknisk
```python
import numpy as np
# [10 linjer kompleks matrix-logik]
```

### ❌ Dårligt: Uden Kontekst
```python
print(x)
```

## Hyppige Tasks

### "Jeg skal tilføje en øvelse til et eksisterende modul"

1. Åbn det relevante notebook
2. Find stedet efter et eksempel
3. Tilføj markdown: `**Øvelse:** [konkret opgave]`
4. Tilføj tomme codecelle for elev-svar
5. Opdater `lærervejledning.md` med løsning

### "Jeg skal lave helt nyt modul"

1. Se SKILL.md Del 5 for checklist
2. Lav `NN_emnenavn.ipynb`
3. Følg struktur: Intro → Teori → Eksempel → Øvelse (×3-4)
4. Lav `NN_emnenavn_løsning.ipynb` med løsninger
5. Tilføj til README.md
6. Opdater `lærervejledning.md`

### "Elever får en fælles fejl"

1. Noter fejlen
2. Tilføj til `lærervejledning.md` under "Fælles Fejl"
3. Lav EKSEMPEL på fejlen i notebooks
4. Vis løsning i løsningsnotebook
