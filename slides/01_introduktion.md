# Modul 01: Introduktion til Programmering 🚀

**Python helt fra bunden**

*Børn uden programmerings-erfaring*

---

## Hvad er programmering?

At **instruere en computer** til at gøre hvad vi vil

```
Hvad programmører gør:
├─ Skriver instruktioner
├─ Computeren følger dem
└─ Resultat: programmer, spil, apps
```

**Eksempel:** Minecraft, Fortnite, Instagram - alle programmeret!

---

## Ada Lovelace - Første Programmør 👩‍💻

- Født 1815 i England
- Skrev første computerkode nogensinde
- **Vigtig:** Programmering var ikke kun for drenge dengang!
- Inspirations-kilde: Vi kan alle programmere

---

## Google Colab - Vores Værktøj

```
Google Colab = Python i browseren
├─ Ingen installation
├─ Kan tilgås fra enhver computer
├─ Virker online
└─ Gemmer automatisk
```

**Vi skriver Python-kode her!**

---

## Vores Først Program

```python
print('Hej verden!')
```

**Hvad sker der?**
- `print()` = besked til computeren
- Computeren printer: `Hej verden!`

---
